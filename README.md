# Project Haruki
The reboot of the discord bot `Haruki` from the python version

---
## What is Haruki
Haruki was a discord bot made back in 2021 using the discord python library. It implement simple commands the helps the users moderate the guild or just to have. Past me did not know how to really code properly, so I am taking this as an opportunity to build, learn and to have fun.

The purpose of this project is to rebuild Haruki in Javascript and integrate an LLM for experimental commands. It uses Ollama as the interface for deploying the LLM.
